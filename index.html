SMART WATER PLANTING USING AUDUINO

Built an Arduino-based automated watering system using a humidity sensor to monitor soil moisture levels and control water flow through pipes. The system is powered by batteries and connected via wires, ensuring efficient plant care with minimal manual intervention. Technologies Used: Arduino microcontroller Humidity and soil moisture sensors Water pump, pipes, and relay module Batteries and connecting wires C/C++ programming for Arduino Highlights: Real-time soil moisture monitoring and automated watering. Energy-efficient design powered by batteries. Scalable for home gardening or small-scale agriculture.
